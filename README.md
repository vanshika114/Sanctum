# Sanctum

**Sanctum** is a minimal productivity web app designed to create a calm, distraction-free workspace.
It combines a Pomodoro timer, a task manager, and motivational quotes in a clean interface to help users stay focused.

The goal of Sanctum is simple: **focus on what matters and block out the noise.**

---

## ✨ Features

* **Pomodoro Timer**

  * Focus, Short Break, and Long Break modes
  * Visual progress ring
  * Adjustable session durations
  * Session tracking with focus indicators

* **Task Manager**

  * Add and delete tasks
  * Mark tasks as complete
  * Filter tasks (All / Active / Done)
  * Clear completed tasks

* **Motivational Quotes**

  * Rotating productivity quotes
  * Manual navigation between quotes

* **Live Clock & Date**

  * Displays the current time and date in the header

* **Theme Toggle**

  * Switch between light and dark themes
  * Theme preference saved in local storage

---

## 🚀 How to Run

1. Download or clone the repository

```bash
git clone https://github.com/your-username/sanctum.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

That’s it. No setup or installation required.
